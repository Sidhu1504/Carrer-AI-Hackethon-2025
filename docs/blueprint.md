# **App Name**: CareerCraft AI

## Core Features:

- Resume Analyzer: Upload a resume in PDF or text format, and the AI tool returns an ATS score and suggests improvements.
- Skill-Gap Analysis: Based on the resume content, the AI tool identifies missing skills and generates a personalized 4-week learning plan.
- Career Self-Evaluation: Users select a profession, and the AI tool generates five profession-specific interview questions.
- Mock Interview: Interactive chat simulates an interview, providing instant AI-driven feedback and scoring on answers.
- Performance Dashboard: Save the interview results in the MySQL database. Present data visually for performance tracking over time, including comparison views.
- Authentication: Enable users to securely sign up and log in to the application using custom JWT-based authentication and bcryptjs password hashing.
- Landing Page: Develop an informative landing page highlighting the 'CareerCraft AI' platform with the contributors details, created by Priyanka Pawar and Shivani Jadhav on December 27, 2025.

## Style Guidelines:

- Dark background with a desaturated blue hue: #22273A for a modern feel and comfortable viewing.
- Primary color: Saturated vibrant blue (#64B5F6) to convey innovation and trust. Suitable for a dark scheme.
- Accent color: Analogous blue-violet (#7986CB) to highlight key interactive elements, such as buttons and links, providing good contrast with the primary color.
- Body and headline font: 'Inter' for a modern, readable interface. 'Inter' is a grotesque-style sans-serif with a modern, machined, objective, neutral look and is suitable for both headlines and body text.
- Lucide-react icons for a consistent and clean visual style.
- Fully responsive layout using Tailwind CSS to ensure a seamless user experience on various devices.
- Subtle transition animations to enhance user engagement without being intrusive.
